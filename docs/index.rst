=====
telix
=====

A Modern telnet client especially designed for BBSs and MUDs.

Built with `telnetlib3 <https://telnetlib3.readthedocs.io/>`_,
`blessed <https://blessed.readthedocs.io/>`_, and
`textual <https://textual.textualize.io/>`_.

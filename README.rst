telix
=====

A Modern telnet client especially designed for BBSs_ and MUDs_.

Built with telnetlib3, blessed, and textual.

.. _BBSs: https://bbs.modem.xyz/
.. _MUDs: https://muds.modem.xyz/

Installation
------------

::

    pip install telix

Usage
-----

::

    telix
